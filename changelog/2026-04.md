# Changelog — April 2026

## 2026-04-14 — Initial Knowledge Base Creation

### Summary
Initial creation of the structured AI music knowledge base from source research files. Converted from raw research documents (PDFs, DOCX) into a structured, model-agnostic knowledge base with model-specific sections.

### Model Versions Covered
- **Suno:** v5.5 (current flagship, March 26, 2026) and v5 (September 2025)
- **Gemini / Lyria:** Lyria 3 Pro (current, March 25, 2026), Lyria 3 Clip, Lyria RealTime

### Files Created

#### Shared (model-agnostic)
- `shared/prompt-framework.md` — Universal prompt structure: [role] + [genre/style] + [arrangement/instrumentation] + [performance/feel] + [mix/space] + [technical constraints]. Full mixing-language vocabulary. Post-production engineer mindset.
- `shared/genre-templates.md` — Reusable genre templates with example prompts for: Americana/Folk/Roots, Country-Soul, Rock (Classic/Power/Alternative/Blues), Trap/Hip-Hop, Orchestral/Cinematic/Classical, Pop/Synth-Pop, R&B/Neo-Soul.
- `shared/iteration-workflow.md` — v1 → v2 → remix workflow. Gibberish/mumble-first technique, section-by-section refinement, hybrid AI+DAW workflows, when to regenerate vs. fix in post.

#### Suno
- `models/suno/overview.md` — Model version history table, core capabilities, platform architecture diagram, strengths/weaknesses, pricing tiers (Free/Pro/Premier), rights/licensing (WMG settlement, DDEX disclosure, copyright situation), safety constraints.
- `models/suno/advanced-prompting.md` — Two-Box Rule, GMIV/5-Signal/4-Part/Post-Production Engineer frameworks, full structural tags reference, vocal performance tags, instructional modifier syntax (parentheses/braces/asterisks), arrangement tags, structure energy tags, punctuation behavior, vocal anchor blocks, exclusion/negative prompting, lyric/prompt bleed prevention, phonetic spelling, Americana/country-soul specifics, best practices.
- `models/suno/hidden-features.md` — Max Mode (with full syntax), Mumble Mode, Sounds Beta Mode, Preserve Melody Tags, Persona Extension Trick, Suno Labs / MILO-1080, Suno Studio, Voices (v5.5), Custom Models (v5.5), My Taste (v5.5), Personas and Covers, Gibberish Vocal Trick. Each item labeled as `official-ui-feature`, `documented-behavior`, or `community-heuristic`.

- `models/gemini-lyria/overview.md` — What is Lyria, relationship to Gemini, model variants table (Clip/Pro/RealTime/legacy), access points (consumer and developer/enterprise surfaces), capabilities and constraints table, SynthID watermarking, licensing and usage rights, active litigation note, strengths and limitations summary.
- `models/gemini-lyria/prompting.md` — Core framework, Anchor Formula, four core rules, prompt elements reference, custom lyrics syntax, timestamp prompting for Lyria 3 Pro (with examples), negative prompts (API), image-to-music inputs, example prompts by complexity level (simple/intermediate/advanced/timestamp), Lyria RealTime parameters (full table), WeightedPrompt system, Composer Mode, what works vs. what doesn't, differences from Suno prompting, use case recommendations table.
